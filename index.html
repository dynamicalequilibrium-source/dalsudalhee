관리자 제어 콘솔

사용자의 입력 텍스트 키워드에 따라 어떤 공식 동작(Action)을 트리거할지 매핑 사전 규칙을 수정합니다.

달서구 캐릭터 바이블의 고정 제약 사항으로, 수정 불가능한 엄격한 규약 항목(Identity Lock) 리스트입니다.

얼굴 구조 고정 (Identity Lock: Face)
달수(가로 타원형, 두 검은 점눈, 귀여운 미소선), 달희(눈 영역 보이지 않거나 감은 눈, 미소선). 두 캐릭터 모두 코와 치아, 눈썹을 그리지 않음.
헤어스타일 규격 (Identity Lock: Hair)
달수(구름 모양 머리, 상단 3개 봉우리, 하단 2개 엽단 배치), 달희(버섯 모양 머리, 앞머리 6가닥). 대칭형 유지.
공식 색상표 잠금 (Identity Lock: Colors)
머리카락(#604C3F), 피부(#EBC8A3), 의상(#B28247), 윤곽선(#231815) Hex 코드가 코드 내부적으로 하드코딩되어 강제 고정됩니다.
비율 및 윤곽 규격 (Identity Lock: Silhouette)
2D 플랫 벡터 렌더링 스타일, 최소한의 2D 명암 처리, 등각 투영(Orthographic) 뷰 고정. 3D 렌더링, 클레이 모델, 웹툰/망가 데포르메 적용을 엄격히 차단합니다.
생성 방식 (Reference Character Mode)
새로운 캐릭터 임의 생성 금지. 공식 원본 파일(기본-달수.png / 기본-달희.png)을 항상 생성 엔진의 Reference Image 슬롯에 바인딩하여 '공식 캐릭터를 참조하여 변형'하는 방식만 수행하도록 제약합니다.
엔진 제어 시스템 프롬프트 (System Prompt)
첨부된 캐릭터를 그대로 유지한다.
얼굴 구조 변경 금지 / 눈 모양 변경 금지
신체 비율 변경 금지 / 복장 변경 금지 / 색상 변경 금지
캐릭터 정체성을 유지하면서 포즈와 상황만 변경한다.

이미지 생성 엔진 해석 단계에서 발생한 캐릭터 붕괴 시도(Drift Detection) 및 자동차단 복구 이력입니다.

현재 감지된 캐릭터 붕괴 위반 시도가 없습니다.
